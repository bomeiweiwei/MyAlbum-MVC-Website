3. 請設計一個跟讀取相關的介面的Repository，名稱是：I<<類別名稱>>ReadRepository
4. 介面的Repository一定有2個方法，設定為：
4-1. 方法一定都會帶：
4-1-1. Req類別，用於帶入linq查詢：
4-1-1-1. 名稱：Get<<類別名稱>>Req
4-1-1-2. 屬性：除了類別屬性的 CreatedAtUtc、UpdatedAtUtc、CreatedBy、UpdatedBy，全都加到Get<<類別名稱>>Req裡
4-1-1-3. 指定的日期欄位：ReleaseTimeUtc
4-1-1-4. 請把屬性全都設為可null，如果指定的日期欄位有設定則新增二個欄位：Start<<指定的日期欄位>>、End<<指定的日期欄位>>
4-1-1-5. 遇到 Status關鍵字 請設定為：public Status? Status { get; set; }，它在系統裡是Enum
4-1-2. CancellationToken ct = default

4-2. 方法一定都會回傳一個Dto，名稱：<<類別名稱>>Dto
4-2-1. <<類別名稱>>Dto的屬性全參照2.的：類別屬性
4-2-1-1. 遇到 Status關鍵字 請設定為：public Status Status { get; set; }，它在系統裡是Enum

4-3. 方法名稱1：Get<<類別名稱>>Async，只取一個Dto物件，可為null

4-4. 方法名稱2：Get<<類別名稱>>ListAsync，取得Dto List
